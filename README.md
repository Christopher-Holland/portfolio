# Developer Portfolio

This is a **responsive, terminal-style portfolio website** showcasing my full-stack projects, technical skills, and personal information. Built with **React**, **TailwindCSS**, and **Vite**, it combines modern design with a minimalist feel.

---

## Features

- Terminal-style UI with smooth animations  
- Responsive design for desktop and mobile  
- Sections for projects, skills, contact, and about me  
- Easy to update and expand with new projects  

---

## Tech Stack

- **Frontend:** React, TailwindCSS, Vite  
- **Deployment:** GitHub Pages  
- **Tools:** Git, GitHub

## Getting Started

### Prerequisites

- Node.js >= 18  
- npm or yarn  

### Installation

1. **Clone the repository:**

bash
git clone https://github.com/Christopher-Holland/portfolio.git
cd portfolio


2. **Install dependencies:**

bash
npm install


3. **Start the development server:**

bash
npm run dev


4. **Open the app in your browser:**  
Visit [http://localhost:5173](http://localhost:5173)
